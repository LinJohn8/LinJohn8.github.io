游戏与插件开发

我开发过很多的插件，一开始是为了能够更好的服务我的游戏，但一言难尽，所以在下面你可以看到我做的很多插件，或许有些已经下架了：

项目模版：

  • Snake Game Template - 一款3D的多人贪吃蛇游戏蓝图模板，有可以操控的贪吃蛇，可以无限繁殖的事物等
  • Simple Rhythm Effect - 一个简单的节奏效果蓝图模板，一个方块的点击从而影响多个方块的项目
  • Easy Change Game Menu - 一个用于游戏的菜单，当时在其中也加入了各种各样的语言翻译，但对于现在而言只是一个老旧是新手产品

插件：

  • MultiThreadPlugins (MTP) - 多线程插件，一个完全是U++制作的插件，是基于人宅这样佬的项目来做的
  • Base Properties Plugin - 基础属性插件，提供一些简单通用的数据结构，这是我当时想要做一个简单游戏时候用的框架
  • Damage Text Anim Plugin - 伤害文字动画插件，基于对象池实现，可以在一秒中有数百个文本伤害的情况下稳定运行
  • AlgorithmsAndMathExtensions - 算法与数学扩展，里面是我添加的一些各种各样的算法
  • QuickAssetOperation (QAO) - 快速资产操作插件，是UE编辑器插件，用于对资产的快速操作
  • ChartExtendPlugins - 图表扩展插件，基于折线图的插件，自定义性强
  • Simple Write Plugin - 简易写入插件，用于快速保存或导出数据，如写为txt，和读取txt
  • Target Tagging Plugin - 目标标记插件，用于识别和标记游戏中的特定对象
